# AI Technical Mentor for Data Roles

Un outil intelligent pour réviser les entretiens techniques en Data Science et Data Engineering.

##  Fonctionnalités
- **Génération dynamique :** Questions renouvelées à chaque session.
- **Analyse de code :** Évaluation de la complexité (Big O) et de la lisibilité.
- **Mode Mentor :** Feedback constructif et résumés de progression.

##  Installation
1. Cloner le repo : `git clone ...`
2. Installer les dépendances : `pip install -r requirements.txt`
3. Créer un fichier `.env` avec votre `GOOGLE_API_KEY`.
4. Lancer : `streamlit run app.py`
